Вывод элементов массива.
Напишите программу, которая выводит все элементы заданного одномерного массива на экран.

Сумма элементов массива.
Реализуйте программу, которая вычисляет сумму всех элементов массива и выводит результат.

Среднее значение элементов массива.
Напишите программу, которая находит среднее арифметическое значений элементов массива.

Нахождение минимального элемента.
Создайте программу для поиска минимального значения в заданном массиве чисел.

Нахождение максимального элемента.
Напишите программу, которая находит максимальное значение в массиве.

Поиск индекса заданного элемента.
Реализуйте программу, которая принимает число и выводит индекс его первого вхождения в массив. Если число не найдено, выведите соответствующее сообщение.

Подсчёт количества вхождений элемента.
Напишите программу, которая считает, сколько раз заданное число встречается в массиве.

Обратный порядок элементов.
Создайте программу, которая выводит элементы массива в обратном порядке.

Переворот массива (реверс).
Реализуйте алгоритм, который изменяет порядок элементов массива на обратный (то есть делает реверс массива).

Сортировка массива (пузырьковый метод).
Напишите программу для сортировки массива чисел методом пузырька.

Сортировка массива с использованием Arrays.sort().
Используйте стандартный метод Arrays.sort() для сортировки массива и выведите отсортированный массив.

Удаление дубликатов.
Реализуйте программу, которая удаляет повторяющиеся элементы из массива и выводит массив с уникальными значениями.
(Подсказка: можно использовать дополнительные структуры данных, например, HashSet.)

Умножение каждого элемента на заданное число.
Напишите программу, которая умножает каждый элемент массива на фиксированное число и выводит изменённый массив.

Копирование массива.
Реализуйте программу, которая создает копию заданного массива и выводит оба массива для сравнения.

Сумма элементов на чётных позициях.
Напишите программу, которая вычисляет сумму элементов массива, находящихся на чётных индексах (0, 2, 4, …).

Нахождение второго по величине элемента.
Создайте программу, которая находит второй по величине элемент в массиве чисел.

Проверка на палиндром (для массива символов или чисел).
Напишите программу, которая проверяет, является ли массив палиндромом (то есть читается одинаково слева направо и справа налево).

Замена отрицательных чисел на 0.
Реализуйте программу, которая заменяет все отрицательные значения в массиве на 0 и выводит изменённый массив.

Сдвиг элементов массива влево.
Напишите программу, которая сдвигает все элементы массива на одну позицию влево, а первый элемент перемещает в конец массива.

Сдвиг элементов массива вправо.
Реализуйте программу, которая сдвигает все элементы массива на одну позицию вправо, а последний элемент перемещает в начало массива.
